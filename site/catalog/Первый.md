# Клубника, Яблоки, Банан, Черника, Ананас, Слива

Это идеальные фрукты для смузи или обычной ассорти нарезки

<img width="740" height="740" alt="image" src="https://github.com/user-attachments/assets/1a0bbeec-e128-4179-a37c-b4855e8bc5e6" />

- Клубника: C , K, B9. Мощный антиоксидант, для иммунитета.

- Яблоки: C, A, плюс много клетчатки. Для пищеварения и бодрости.

- Банан: B6, калий. Для энергии, сердца и хорошего настроения.

- Черника: K, C, огромное количество антиоксидантов. Для зрения, молодости и сосудов.

- Ананас: C, B1, B6. Для иммунитета и пищеварения (благодаря бромелайну).

- Слива: C, A, K. Для мягкого очищения кишечника и здоровья костей.
